In this package, all the basic operations on shared secrets are implemented. For a detailed specification of the operations, please refer to the SEPIA Technical Report.

The implementation of the LessThan operation and all its suboperations is based on the following paper which is references as in the documentation:
  • [Nishide2007] T. Nishide and K. Ohta. Multiparty computation for interval, equality, and comparison without bit-decomposition protocol. In Conference on Theory and Practice of Public Key Cryptography (PKC), 2007.
